# cola_project_Public_Affairs
โปรเจคนี้เป็นส่วนหนึ่งของรายวิชา CP413601

**ความเป็นมา**

รายงาน World Population Prospects 2022 ระบุว่า ประชากรโลกมีจำนวนสูงถึง 8 พันล้านคน โดยผู้ที่มีอายุมากกว่า 65 ปีอยู่ราว 10% และคาดว่าจะเพิ่มขึ้นเป็น 16% ภายในปี 2050 ขณะเดียวกันจำนวนผู้ที่มีอายุ 80 ปีขึ้นไป จะเพิ่มจาก 157 ล้านคนเป็น 459 ล้านคน ภายใน 30 ปีข้างหน้า สะท้อนแนวโน้มการเข้าสู่สังคมสูงวัยของโลกอย่างชัดเจน

สำหรับประเทศไทย รายงานระบบสถิติทางการทะเบียนระบุว่าในปี 2022 ประเทศไทยได้ก้าวเข้าสู่ สังคมผู้สูงอายุอย่างสมบูรณ์ โดยมีสัดส่วนประชากรอายุ 60 ปีขึ้นไปคิดเป็น 19.97% ของประชากรทั้งหมด

หนึ่งในปัญหาสำคัญที่กระทบคุณภาพชีวิตผู้สูงอายุคือ การพลัดตกหรือหกล้ม ซึ่งเป็นสาเหตุการบาดเจ็บรุนแรงและการเสียชีวิตที่พบได้บ่อย โดยข้อมูลจากกรมควบคุมโรคในปี 2567 ระบุว่า มีผู้สูงอายุเสียชีวิตจากการพลัดตกจำนวน 1,504 คน มีผู้ที่ต้องเข้ารับการรักษาในโรงพยาบาลถึง 85,774 คน และสร้างภาระค่าใช้จ่ายรวมมากกว่า 2,600 ล้านบาท อีกทั้งยังมีแนวโน้มเพิ่มสูงขึ้นอย่างต่อเนื่อง

ด้วยสถิติที่เพิ่มขึ้นทุกปี การมีระบบหรือเทคโนโลยีที่ช่วย เฝ้าระวัง คอยตรวจจับ และแจ้งเตือนการหกล้ม จึงเป็นสิ่งที่มีความจำเป็นอย่างยิ่ง ไม่เพียงช่วยลดความสูญเสีย แต่ยังช่วยเพิ่มความปลอดภัยและคุณภาพชีวิตของผู้สูงอายุในระยะยาวอีกด้วย


**วัตถุประสงค์ของโครงการ (Objectives)**


1.เพื่อตรวจจับการพลัดตก–หกล้มได้อย่างแม่นยำและรวดเร็วมากขึ้น
2.เพื่อลดอัตราการบาดเจ็บรุนแรงและการเสียชีวิตของผู้สูงอายุ โดยเพิ่มโอกาสได้รับความช่วยเหลือทันที


**ขอบเขตของโครงการ (Scope)**


1.พัฒนาโมเดลตรวจจับท่าทาง (Pose Estimation) หรือการจำแนกการกระทำ (Action Recognition) จากวิดีโอผ่านกล้องวงจรปิด (CCTV/Webcam)
2.การประมวลผล ระบบสามารถวิเคราะห์ภาพจากกล้องแบบ Real-time และระบุสถานะ "การหกล้ม" ได้
3.การแจ้งเตือน เมื่อตรวจพบการหกล้ม ระบบจะส่งข้อความแจ้งเตือนพร้อมภาพหลักฐานไปยัง API (เช่น Line Notify) เพื่อแจ้งผู้ดูแล


**ผู้มีส่วนได้ส่วนเสีย (Stakeholder Analysis)**


1.ผู้สูงอายุและผู้ป่วย:กลุ่มเป้าหมายหลักของโครงการ ผู้ได้รับประโยชน์โดยตรงในด้านความปลอดภัยและการเข้าถึงความช่วยเหลือที่รวดเร็ว
2.ครอบครัวและผู้ดูแล: ลดภาระความกังวลและช่วยให้ดูแลผู้สูงอายุได้มีประสิทธิภาพมากขึ้นแม้ไม่ได้อยู่ด้วยกันตลอดเวลา
3.หน่วยงานสาธารณสุข/ศูนย์กู้ชีพ: ได้รับการแจ้งเหตุที่แม่นยำ ช่วยให้การบริหารจัดการรถพยาบาลและการส่งต่อผู้ป่วยทำได้ทันท่วงที
4.องค์กรปกครองส่วนท้องถิ่น: เช่น เทศบาล หรือ อบต. ที่สามารถนำระบบไปใช้ในศูนย์พักพิงหรือบ้านพักผู้สูงอายุในชุมชน


**ประเด็นทางนโยบาย (Policy Issues)**


1.การรองรับสังคมผู้สูงอายุอย่างสมบูรณ์
2.การเข้าถึงเทคโนโลยีเพื่อความปลอดภัยของผู้สูงอายุ
3.งบประมาณและการลงทุนด้านสาธารณสุข
4.มาตรฐานความปลอดภัยของผู้สูงอายุในบ้านพักและสถานดูแล
5.ความเป็นส่วนตัวของข้อมูล(PDPA) การใช้กล้องตรวจจับต้องคำนึงถึงสิทธิความเป็นส่วนตัว และมีมาตรการคุ้มครองข้อมูลภาพส่วนบุคคล


**การบริการสาธารณะ (Public Services)**


1.ระบบแจ้งเตือนฉุกเฉิน (Emergency Alert System)
2.การดูแลสุขภาพทางไกล (Telehealth/Telecare)
3.ระบบติดตามสุขภาพผู้สูงอายุ (Health Monitoring System)
4.ลดจำนวนเหตุฉุกเฉิน ที่เสี่ยงต่อชีวิตและงบประมาณรักษาพยาบาล
